使用步骤：

```
chmod +x ./install.sh
sudo ./install.sh
```

说明：

1、运行安装脚本需要root权限。

2、使用solarized时，需要对Terminal进行一些配置，查找路径是：
```
Edit -> Preferneces -> profiles -> Edit -> Colors
```
配置如下：
```
Text and Background Color
- 关闭 Use colors from system theme
- Built-in schemes 选择Solarized dark

Palette
- Built-in schemes 选择Solarized
```
